html attribute

href attribute
href locate url reference for <a>
<a> add the url link to a text
tag elements

<a href = "url link">url link name</a>

example code

<a href = "https://www.google.com" >google url <a>

result

google url

src and alt attribute
src locate image file for <img>
alt alternative name for image when image unavaliable
<img> add image to html page
tag elements

<img src = "image directory" alt = "alternative name" >

example code #1

<img src = "image/image.jpg">

result

example code #2

<img src = "picture.jpg" alt="an image"> #there was no file name picture.jpg

result

width and height attribute
both width and height use to set an image size
tag elements

<img src = "image/image.jpg" width = "width" height = "height">

example code

<img src = "image/image.jpg"> width = "512" height = "288">

result

title attribute
title use to show more infomation for the paragraph when the cursor over the text
tag elements

<p title = "text infomation">paragraph</p>

example code

<p title = "just a text">get the cursor over me</p>

result

get the cursor over me
