html tutorial

html elements

<start tag>tag content</end tag>
tag เป็นตัวที่ใช้บอกชนิดของ tag content

html attribute

href attribute
href มีใว้สำหรับระบุ url ที่ใว้อ้างอิงของ tag <a>
<a> ใช้ในการเพิ่ม url ให้กับตัว text
tag syntax

<a href = "url link">url link name</a>

example

<a href = "https://www.google.com" >google url <a>

result

google url

src and alt attribute
src ใช้ระบุตำเเหน่งของไฟล์ให้กับ <img>
alt ใช้เพื่อกำหนดชื่อของรูปภาพเมื่อไม่สามารถเเสดงรูปใด้
<img> ใช้เพื่อเพิ่มรูปภาพในหน้าของ html
tag syntax

<img src = "image directory" alt = "alternative name" >

example #1

<img src = "image/image.jpg">

result

example #2

<img src = "picture.jpg" alt="an image"> #รูปนี้ไม่มีอยู่มีอยู่บนเว็บ

result

width and height attribute
ทั้ง width and height ใช้สำหรับกำหนดขนาของรูปภาพที่เเสดงบน html
tag syntax

<img src = "image/image.jpg" width = "width" height = "height">

example code

<img src = "image/image.jpg"> width = "512" height = "288">

result

title attribute
title ใช้เพื่อเเสดงข้อมูลเพื่มเติมเมื่อนำ cursor วางใว้บน text
tag syntax

<p title = "text infomation">paragraph</p>

example

<p title = "just a text">get the cursor over me</p>

result

get the cursor over me

style attribute
style ใช้สำหรับปรับเเต่งสีเเละขนาดของ text
tag syntax

<p style = "color:any color or rgb values;">paragraph with color</p>

example #1

<p style = "color:blue;">paragraph with color</p>

result

paragraph with color

example #2

<p style = "color:rgb(241, 165, 50);font-size:30px">paragraph with color and bigger text</p>

result

paragraph with color and bigger text

lang attribute
<html lang = "language code"> lang ใช้เพือระบุภาษาของหน้า web page เพื่อใช้กับ search engine

html heading

heading จะสามารถเเบ่งใด้จาก <h1> ถึง <h6> เมื่อ h1 เป็นหัวข้อที่สำคัญที่สุดเเละ h6 เป็นหัวข้อที่สำคัญน้อยที่สุด

example

<h1>1 heading</h1>
<h2>2 heading</h2>
<h3>3 heading</h3>
<h4>4 heading</h4>
<h5>5 heading</h5>
<h6>6 heading</h6>

result

1 heading

2 heading

3 heading

4 heading

5 heading
6 heading

นอกจากนี้สามารถปรับเเต่ง heading โดยใช้ style ใด้ด้วย

example code

<h1 style = "font-size:40px;color:lightsalmon;">custom heading</h1>

customize heading

html paragraph

paragraph ใช้กับตัวเนื้อภายใน html

ตัวของ paragraph เมื่อทำการประกาศใช้จะเป็นการขึ้นบรรทัดใหม่ทุกรอบที่ประกาศใช้
example

<p>first paragraph</p>
<p>second paragraph</p>
<p>thrid paragraph</p>

result

first paragraph

second paragraph

thrid paragraph

ตัวของ paragraph นั้นจะไม่สนว่าตัว text นั้น จะอยู่ห่างจากกันเท่าใหร่หรือว่าจะอยู่คนละบรรทัดก็จะนำมาเเสดงเป็นอันเดียวกัน
example

<p>this          is the 
                same paragraph</p>

result

this is the same paragraph

เนื่องจาก <p> นั้นไม่สนเรืองบรรทัดเเละระยะห่างของตัว text จึงทำให้การ quote คำพูดหรือ text ที่ใด้มีการจัดรูปใว้เเล้วนั้นทำใด้ยากจึงใด้มี <pre> เพื่อใว้ใช้กับบทความที่มีการจัดเรียงมาใว้เเล้ว
example code

<pre>"There is no law except the law that there is no law." 
                - John Archibald Wheeler</pre>
"There is no law except the law that there is no law." 
                            - John Archibald Wheeler

html style

style ใช้สำหรับปรับเเต่งสี ขนาด font เเละ อื่นๆ ของ tag ต่างๆ
tag syntax

<tagname style="property:value;">tag content</tagname>

background-color
background-color เป็นการกำหนดสีพื้นหลังของ tag ที่ใช้กับ style
example

<p style = "background-color:coral">a text with changing background-color</p>

result

a text with changing background-color

color
color เป็นการกำหนดสีของตัวอักษรที่เเสดง
example

<p style="color: indigo;">colored text</p>

result

colored text

font-family
font เป็นการกำหนด font ของตัวอักษรที่เเสดง
example

<p style="font-family: fantasy;">this is fantasy font</p>

result

this is fantasy font

font-size
เป็นการกำหนดขนาดของตัวอักษรที่เเสดง
example

<p style="font-size:200%;">bigger paragraph</p>

result

bigger paragraph

text-align
เป็นการกำหนดตำเเหน่งของตัวอักษรที่เเสดง
example

<p style="text-align: end;">text at the most right</p>

result

text at the most right
